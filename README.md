## Web Development Tasks Repository

This repository contains Web Development tasks and assignments completed as part of my Computer Science coursework.
Each task focuses on practicing core front-end concepts using HTML5 and CSS3.

# Repository Purpose

The purpose of this repository is to:

Practice fundamental web development concepts

Apply semantic HTML and structured CSS

Build responsive and user-friendly web pages

Maintain organized task-based project structure

# Tasks Overview
Task 1: Art Gallery Website

Description:
A responsive single-page Art Gallery website designed using HTML5 and CSS only.
The website includes multiple sections such as gallery, about, sign-up, and feedback, all implemented within one HTML file and one external CSS file.

Key Concepts Used:

Semantic HTML5 elements (header, nav, section, footer)

External CSS styling

Navigation bar with pseudo-class hover effects

Image gallery using Flexbox/Grid

Styled forms (Sign Up & Feedback)

Google Fonts for typography

Font Awesome icons

Responsive design using media queries
